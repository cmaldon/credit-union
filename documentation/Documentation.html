Fina is a Modern responsive HTML Template. It is especially designed for Business, Financial Advisor, Accountant, Law Firm, Wealth Advisor, Investment and general corporate website. This theme comes with 4 homepage variant, this template is robust and easy to modified because it uses UIkit framework.

HTML Structure

The layout in Fina is flexible because this use "grid" system based on uikit, so you can make any configuration of column (please refer the "https://getuikit.com/docs/width" for some sample of column). Navigation menu is within a nav with an class of "uk-navbar-container". The general template structure is the same throughout the template. Here is the general structure :

Site Structure

The page have five main blocks : header, pagetitle, content-area, and footer each block consisting of each element present in it.

Logo

To edit the logo you can simply replace the "logo.svg" for header logo file into the images folder, if you intend to use a png file or svg file. Default size for header logo is 140pixel for width.

You can change with another big size logo, but it's not recommended and offcourse make the design not good :-)

Menu

Menu at this theme is use horizontal menu style, and it use nav with ul class "uk-navbar-nav" and for dropdown just make div class "uk-navbar-dropdown" between parent li menu. If you want make active some menu please use class="uk-active" in active menu that you want. See the image below to understand how it work.

How to change Map location ?

You must set "latitude" point and "longitude" point, to know your position coordinates or latitude and longitude please visit : http://www.mapcoordinates.net/en

Change that value on the file config-contact.js (js/components/config-contact.js) also don't forget change the latitude and longitude for marker icon with your value.

How to change or add stock ticker

You can change the default stock ticker that display on homepage 1. Edit the current stock ticker with anything do you like, the ticker is based on Yahoo Finance so it recommend to see stock code from there.

Change that value on the file config-stock-ticker.js (js/components/config-stock-ticker.js).

How to setting Contact Form ?

To setting contact form, please open file "sendmail.php" and change with your email on the line 7, also you can change the email subject on the line 9.

How to make a new Favicon ?

First you must make image on size 32x32 pixel, and save in png format, after that go to this site : http://tools.dynamicdrive.com/favicon/ and create your "favicon.ico" there and replace the old "favicon.ico" in image folder with your new favicon.ico
